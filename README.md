# DiffScope Synthesis Platform

The synthesis backend for DiffScope (under development)

## Prerequisites

- CMake
- Go
- SWIG
- GCC/Clang (MSVC is not supported due to Cgo limitations)

## Building

Use CMake to configure and build the project. The Go code will be built as part of the CMake build process.
